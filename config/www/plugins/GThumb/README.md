# GThumb Plugin for Piwigo

A gallery thumbnail plugin for Piwigo that provides enhanced thumbnail display functionality.

## Features

- Enhanced thumbnail gallery display
- Image loading optimization
- Responsive design support

## Installation

1. Copy the `GThumb` folder to your Piwigo `plugins` directory
2. Go to Administration > Plugins
3. Activate the "GThumb" plugin

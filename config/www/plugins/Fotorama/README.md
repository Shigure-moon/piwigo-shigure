# Fotorama Plugin for Piwigo

A slideshow plugin for Piwigo using the Fotorama library for beautiful image galleries and slideshows.

## Features

- Beautiful slideshow interface
- Touch-friendly navigation
- Full-screen mode support
- Responsive design
- PhotoSwipe integration option

## Installation

1. Copy the `Fotorama` folder to your Piwigo `plugins` directory
2. Go to Administration > Plugins
3. Activate the "Fotorama" plugin

## Usage

Once activated, the plugin will provide enhanced slideshow functionality for your Piwigo gallery.

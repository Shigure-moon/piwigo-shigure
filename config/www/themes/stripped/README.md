# Stripped Theme for Piwigo

A modern, clean theme for Piwigo focused on showcasing images with a premium design.

## Features

- **Modern Gallery Design**: Waterfall/Justified layout with smooth animations
- **Clean Interface**: Minimalist design that puts focus on your images
- **Responsive**: Works beautifully on all devices
- **Customizable**: Modern Gallery Settings for layout customization
- **Navigation**: Clean navigation buttons matching the top navbar design

## Installation

1. Copy the `stripped` folder to your Piwigo `themes` directory
2. Go to Administration > Configuration > Themes
3. Select "stripped" theme
4. Configure your settings in the Modern Gallery Settings section

## Configuration

The theme includes a simplified configuration interface with:

- **Modern Gallery Settings**: 
  - Number of random photos per page
  - Waterfall columns
  - Gap between items
  - Minimum column width
  - Layout direction (Row/Column)
  - Target row height
  - Custom CSS

## Customization

You can add custom CSS in the Modern Gallery Settings section, or create a `local/custom.css` file in the theme directory.
